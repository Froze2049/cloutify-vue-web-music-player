# cloutify-vue-web-music-player

基于网易云 API 搭建的 web 音乐播放器，使用技术栈：Vue3，Vue-Router，Vuex，Axios，Element-Plus。

使用前请先安装并运行 网易云音乐 Node.js API service
https://github.com/Binaryify/NeteaseCloudMusicApi

### 项目安装

```
npm install
```

### 项目运行

```
npm run serve
```
