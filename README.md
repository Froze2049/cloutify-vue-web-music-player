# cloutify-vue-web-music-player

基于网易云 API 搭建的 web 音乐播放器，使用技术栈：Vue3，Vue-Router，Vuex，Axios，Element-Plus。

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
